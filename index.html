Transistor-based Logic

All the computer hardware we know and love today is based on the semiconductor transistor .

Semi-conductors in a hurry!

Transistors = Semi-Conductor Juncitons

With these devices, we can create logic gates.

Example: AND Gate

A B OUTPUT
0 0 0
1 0 0
0 1 0
1 1 1

many possible logic gates can be created

  • NAND gates are universal .
  • Combine transistors to make NAND gates. Combine NAND gates for all logical operations.
